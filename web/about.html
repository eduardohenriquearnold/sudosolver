About

SudoSolver was created by Eduardo Henrique Arnold as a C++ project. With time it was enhanced to allow image based queries and to solve more complex puzzles. Furthermore, it allows the user to generate new puzzles.

After developing the C++ application this web wrapper was designed to facilitate user iteration.

Next I plan to improve the base application to take advantage of multiple cores processing.
